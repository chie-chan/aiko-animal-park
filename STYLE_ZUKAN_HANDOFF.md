# うちの子AIスタイル図鑑 ― 整理リファクタ引き継ぎメモ

> 作成日：2026-05-12
> 対象：Codex（後任エージェント）
> 目的：N系95枚を含む現状のスタイル図鑑を、お客様が選びやすい構造へ整理する

---

## 1. 背景（なぜ整理が必要か）

### 経緯
- もともと制作実績ベースの「ギャラリースタイル一覧」（W系・FP・FS・D・E・C系）だけだった
- **2026-05-12 に N系95枚を一気に追加**（note由来のスタイルカタログ）
- これにより図鑑が**急に大きくなり、カテゴリ構造が破綻**
- お客様から「**スタイル一覧が見えない／何があるか分からない**」という声

### ビジネスコンテキスト（あいこさんからのヒアリング）
- **図鑑の目的＝両方**：①注文誘導 ②ブランディング
- **想定する図鑑のお客様**：オーダー商品を「自分でスタイル選んで注文する」人
  - 対象商品：ポスター（A4・L判・スクエア）／デジタルアート／ハンカチオーダー／フローラルポートレート／3Dミニチュア／フレークシール 等
  - ※ワンコイン商品はあいこさんがデザイン固定（既製品）→ 図鑑には**出さない**
- **戦略の方向性**：各画風を均等に並べる。ただし**迷う人には「水彩」を推す**救済導線あり
- **プロンプトは図鑑に出さない**。サムネと一言説明だけで選んでもらう

### 制約（重要）
- ⚠️ **N系などスタイルのID（N069等）はプロンプト辞書（制作スタジオ）と紐付いている。絶対に変更しない**
- ⚠️ `category` フィールドも制作スタジオと整合しているため、原則そのまま
- ⚠️ プロンプト本文を図鑑画面に出してはいけない（公開ガイドライン）

## 2. 現状の問題点

### 問題①：カテゴリの判断軸が混在している
現状15カテゴリのうち、軸がバラバラ:

| 軸 | 該当 |
|---|---|
| **画風**（どう描くか） | 水彩・刺繍・ステンドグラス・モダン・クラフト |
| **テーマ**（何を描くか） | お花・スイーツ・カフェ・バー・記念日 |
| **雰囲気** | レトロ・ポップ・ファンタジー・リラックス |
| **表現形態** | スノードーム・着せ替え |

→ 「水彩のお花」「刺繍の記念日」のような**重複指定したい作品が片方に偏ってしまう**。

### 問題②：「全体を見渡す入口」がない
- 現状、ページを開いてもカテゴリを選ぶまで何も表示されない
- 「すべて」ボタンはあるが**最後に追いやられていて気づかない**
- お客様の主訴「**スタイル一覧が見えない**」の主因

### 問題③：水彩の細分化ができていない
- 水彩5枚全部「N069〜N073 ノスタルジック水彩」シリーズ（クラシック系のみ）
- 今後増やしたい：コテージコア水彩、ふんわり水彩、ヴィンテージカード水彩 等
- 現状の構造だと**全部「水彩」フラットに並んで区別できない**

### 問題④：用途（ポスター向き／ハンカチ向き等）が示されていない
- 同じ水彩でも「ポスターに耐える緻密さ」と「ハンカチに乗せやすいシンプルさ」がある
- お客様は商品に応じてスタイルを選びたい
- 現状、カードを見ても**どの商品に合うか判断材料がない**

## 3. 整理の方針（あいこさんとの合意事項）

### 方針A：画風8つでメインタブを構成（フラット均等）
判断軸を「画風」に統一する:

```
1. 水彩
2. 刺繍
3. ステンドグラス
4. スノードーム
5. フローラルポートレート
6. 3Dミニチュア・ジオラマ
7. 花冠・ポスター（フォト系）
8. レトロ・ポップ
9. ファンタジー・着せ替え（任意）
10. クラフト・モダン（任意）
```

「お花」「記念日」「スイーツ」「カフェ・バー」「リラックス」は **画風ではないので画風タブから外す**。テーマタグに降格。

### 方針B：水彩の内部細分化（subCategory）
水彩は今後の主役カテゴリなので、4〜5つのサブカテゴリで構造化:

```
水彩
├─ クラシック       … N069〜073（ノスタルジック・写実）
├─ コテージコア      … 苺バスケット系（W-007 等）
├─ ふんわり         … 絵本イラスト風（W-002 等）
├─ ボタニカル       … お花重視（W-004 ラナンキュラス等）
└─ ヴィンテージカード … 古いポストカード風
```

### 方針C：テーマタグ（横断、複数付与可）
画風と直交する軸として:

```
お花 / 記念日 / スイーツ / カフェ・バー / リラックス /
ノスタルジー / アウトドア / 季節 / 着せ替え
```

### 方針D：用途タグ（販売との連動、複数付与可）
お客様の「何に使うか」目線:

```
🖼 ポスター向き（A4・L判・大判印刷で映える）
📱 デジタル向き（壁紙・SNSヘッダー）
🎁 ギフト向き（記念日・誕生日感）
🌸 SNS映え（インスタ・X 向き）
🧺 グッズ向き（ハンカチ・トートに合う）
```

※「ワンコイン向き」は不要（お客様は選ばないため）

### 方針E：迷う人への救済「水彩推し」を1か所だけ
- メインタブはフラット均等
- ヒーロー部分に**さらりと一言**「迷ったら水彩がオールマイティ🌷」と添える
- 押しつけにならないバランス

## 4. 推奨実装ステップ

### Phase 1【最優先・1日】：表示層だけの組み替え

1. `StyleZukanPage.tsx` の `CATEGORY_GROUPS` を新8〜10カテゴリに組み替え
2. 「すべて」ボタンを **先頭** に置く（現状最後）
3. 初期表示で「すべて」または「水彩」をアクティブにする（ページ開いた瞬間に一覧が見える）
4. 各カテゴリ表示時に**枚数**を出す：「水彩（5）」など（既存 `categoryCounts` を活用）
5. ヒーロー部分に「迷ったら水彩」の救済テキスト追加

→ **ID・データに一切手を入れない、CSS と JSX のみ修正**で済む。

### Phase 2【中規模・2〜3日】：データ拡張

1. `StyleZukanEntry` 型に以下のフィールドを追加（任意プロパティ）:
   ```ts
   type StyleZukanEntry = {
     // 既存（変更しない）
     id: string;
     category: string;
     ...
     
     // 新規追加（オプショナル）
     subCategory?: string;       // 水彩内の細分化用「クラシック」「コテージコア」等
     themes?: string[];          // ["お花", "記念日"]
     usages?: string[];          // ["ポスター向き", "ギフト向き"]
     displayCategory?: string;   // 表示用の再分類カテゴリ（裏側 category は触らない）
   };
   ```
2. `styleZukanGeneratedData.ts` の95枚＋αに `displayCategory` / `themes` / `usages` を手動で付与
   - **既存の `category` フィールドは絶対に変更しない**（制作スタジオと紐付き）
3. UI に「テーマで絞る」「用途で絞る」のサブフィルター行を追加
4. 各カードに「向いてる用途」アイコンを表示

### Phase 3【継続】：拡張

1. 水彩のサブカテゴリページ（「水彩」を選ぶと「クラシック / コテージコア / ふんわり / ヴィンテージ / ボタニカル」のセカンダリタブ）
2. 刺繍など他カテゴリも内部細分化
3. note 商材ページ（水彩シリーズ、刺繍シリーズ等）への動線を作成
